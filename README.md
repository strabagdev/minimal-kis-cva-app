# Pulseframe

Pulseframe is a minimal techno pattern and arrangement tool built with Tauri, React, TypeScript, Zustand, and Tone.js.

## KIS-CVA

Keep It Simple.

Create. Variate. Arrange.

Pulseframe has two main areas:

- Pattern Studio: create and modify complete grooves.
- Arrangement: build a song by arranging existing patterns.

## Development

Install dependencies:

```powershell
npm.cmd install
```

Run the desktop app:

```powershell
npm.cmd run tauri:dev
```

Validate:

```powershell
npm.cmd run build
cd src-tauri
cargo check
```
