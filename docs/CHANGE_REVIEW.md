# Change Review

Every future change must answer:

1. Does it help CREATE?
2. Does it help VARIATE?
3. Does it help ARRANGE?

If it does not help any of these:

Do not implement it.

Also ask:

- Is there a simpler solution?
- Does this introduce a new concept?
- Is it really necessary?
